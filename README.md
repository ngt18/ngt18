## Sobre mim

Sou um desenvolvedor em constante evolução, focado em construir aplicações web e mobile com código limpo e bem testado. Tenho maior experiência prática com **JavaScript**, **TypeScript**, **React** e **Node.js**, e venho expandindo meu conhecimento para automações, banco de dados, arquitetura de APIs e desenvolvimento mobile nativo.

Gosto de transformar ideias em projetos reais: desde APIs REST completas e interfaces modernas até automações que integram diferentes serviços e dispositivos.

---

## Áreas de atuação

- **Frontend web** — interfaces responsivas com React, Vite, HTML, CSS e JavaScript vanilla
- **Backend e APIs REST** — Node.js, Express, NestJS, autenticação JWT e bancos de dados relacionais
- **Automação e integração** — leitura e processamento de dados com Python, APIs externas e exportação de relatórios
- **Mobile nativo** — aplicativos Android com Kotlin, Jetpack Compose e arquitetura MVVM
- **Testes automatizados** — testes unitários com Jest e testes E2E com Cypress
